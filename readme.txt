Prima di eseguire il programma, nel caso si voglia essere riconosciuti, è necessario:
- Aggiungere nella cartella dataset una nuova cartella chiamata Nome_Cognome
- Aggiungere delle proprie foto (il numero consigliato è 5) nella cartella appena creata
Se si desidera inserire una password personale
- Aggiungere nella cartella password un file di testo con lo stesso nome della cartella creata in dataset (Nome_Cognome)
- Inserire nel file di testo una stringa di soli 1 e 2. 
 ( 1 rappresenta occhiolino sinistro, 2 rappresenta occhiolino destro. )
